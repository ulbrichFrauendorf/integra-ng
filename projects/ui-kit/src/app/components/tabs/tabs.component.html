Getting Started

Import the ITabs and ITabPanel components in your module or component:

  • Use <i-tabs> as the container element
  • Add <i-tab-panel> for each tab
  • Set header for text labels
  • Set icon for icon display (using PrimeIcons)
  • Use [(activeIndex)] for two-way binding
