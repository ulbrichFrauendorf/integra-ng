Theming

Learn how to customize the appearance of Integra-NG components using CSS variables for light and dark themes.

Integra-NG uses CSS variables (custom properties) for theming. You define variables for both .light and .dark classes, then add the appropriate class to your page's <body> element.

All components automatically use these CSS variables, so changing the theme is as simple as switching the body class!

Here's a breakdown of the theme variables:

Color Variables

  • --color-primary: Main brand color
  • --color-success: Success state color (green)
  • --color-info: Info state color (blue)
  • --color-warning: Warning state color (yellow)
  • --color-danger: Danger state color (red)
  • --color-contrast: High contrast color
  • --color-contrast-inverse: Inverse of contrast color

Text Colors

  • --color-text-primary: Main text color
  • --color-text-secondary: Secondary text color
  • --color-text-tertiary: Tertiary text color
  • --color-text-contrast: Text on contrast backgrounds

Component Background Colors

  • --color-component-background: Primary component background
  • --color-component-background-secondary: Secondary component background
  • --color-component-background-solid: Solid component background

Border Colors

  • --color-border: Component border color
  • --surface-border: Surface border color

Surface Colors

  • --surface-ground: Page background
  • --surface-elevated: Elevated surface background
  • --surface-section: Section background
  • --surface-card: Card backgrounds
  • --surface-overlay: Overlay/dropdown backgrounds
  • --surface-hover: Hover state background
Pro Tip: You can customize these colors to match your brand. Just make sure to keep the variable names the same!
