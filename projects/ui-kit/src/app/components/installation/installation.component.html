Installation

Get started with Integra-NG by installing the library and importing the components you need.

Install the Integra-NG library using npm. Run this command in your project directory:

Import the components you want to use in your Angular component. All Integra-NG components are standalone and can be imported directly.

Integra-NG uses Angular standalone components, so you don't need to import an NgModule.

Now you can use the imported components in your template. Here's a simple example:

You can import multiple components at once from the library:
  • Angular: Version 18 or higher
  • PrimeIcons: Included with the library for icons
  • TypeScript: Version 5.0 or higher
That's it! You're ready to start building with Integra-NG.
