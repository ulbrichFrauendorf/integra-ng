Table Component

A single-input table driven by GridData<TRow, TDetail>. Pass columns, rows, per-row actions, and an optional detail sub-table in one object. Use [virtualScroll]="true" for large datasets (≥500 rows) to enable CDK virtual rendering.

Click column headers to sort. Use the global search bar or per-column filter inputs to narrow results. Uses a regular overflow-scroll table; for 10 000+ rows use [virtualScroll]="true".

Actions are embedded directly in GridData.actions — each carries its own handler. Delete is disabled for out-of-stock rows. Selected: {{ selectedProducts.length }}

Set GridData.details to replace grouped mode. The parent rows show category summaries; expanding a row renders a nested <i-table> with the matching products and their own actions.

Use type: 'icon' with iconClass and severity for colour-coded status icons. Use type: 'list' to render array values as <i-chip> tags.

10 000 rows rendered with CDK virtual scroll — only visible rows are in the DOM. Enable [downloadable] for client-side CSV / JSON / Excel export.
